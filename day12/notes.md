# multi index and pivot tables
what is multi index 
- when we group a data using more than one columns , pandas creates a hierarchial index called a multi index 

what is pivot table 
- a pivot table rearranges data into rows and columns for easier comparison 
rows - ethnic class 
columns - gender 
values- mathscore 
aggregation - maximum 

# diff between group by and pivot 
groupby
- summarized statistics
-better for calculation 
-output usually vertical

pivot table 
- reshapes data
- easier to read
-similar to excel pivot tables
