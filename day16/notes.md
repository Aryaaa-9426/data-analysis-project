| Function               | Purpose                      
| `isna()` / `isnull()`  | Detect missing values                                     |
| `sum()`                | Count missing values in each column                       |
| `fillna(value)`        | Replace missing values                                    |
| `median()`             | Find the median of a numeric column                       |
| `dropna()`             | Remove rows/columns with missing values                   |
| `dropna(subset=[...])` | Remove rows where specified column(s) have missing values |
| `pd.concat()`          | Combine DataFrames by rows or columns                     |
| `pd.merge()`           | Join DataFrames using a common key                        |
| `how="left"`           | Keep all rows from the left DataFrame                     |
| `how="right"`          | Keep all rows from the right DataFrame                    |
| `how="inner"`          | Keep only matching rows                                   |
| `how="outer"`          | Keep all rows from both DataFrames                        |
